# Password_Checker
A software tool that notifies the user whether the entered password is strong or weak based on different criteria such as length,  character, complexity. I have used Python language. 

**#Commands**<br>
Installing all the required libraries <br>
pip install -r requirements.txt<br>

For Executing the main file<br>
python app.py<br>
